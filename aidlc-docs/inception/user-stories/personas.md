## Persona 1: 文化探索者（访客）

| 属性 | 描述 |
|------|------|
| **角色** | Guest（访客） |
| **代表人物** | Sarah，28岁，美国文化研究专业研究生 |
| **语言** | 英文为主，基础中文 |
| **目标** | 了解中国非物质文化遗产，为学术研究收集资料 |
| **动机** | 对东方文化的好奇心，学术研究需要，旅行前了解当地文化 |
| **痛点** | 现有资料分散、语言障碍、缺乏系统化的非遗信息来源 |
| **使用场景** | 通过搜索引擎发现网站 → 浏览非遗分类 → 查看项目详情 → 观看视频 |
| **设备** | 笔记本电脑、平板 |
| **频率** | 每周 2-3 次，每次 20-40 分钟 |
