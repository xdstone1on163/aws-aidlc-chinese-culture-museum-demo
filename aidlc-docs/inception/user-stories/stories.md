# 非遗博物馆 - 用户故事

> 组织方式：按用户角色 | 粒度：中等（~40个故事） | 验收标准：标准型（4-6条）
> 优先级标签：`[MVP]` = P0 首期交付 | `[P1]` = 第二迭代 | `[P2]` = 远期规划

---

## 一、访客（Guest）故事

### US-G01: 浏览非遗项目列表 `[MVP]`
**作为** 访客，**我想要** 浏览非遗项目列表，**以便** 快速了解平台上有哪些非遗项目。

**验收标准：**
1. 首页展示非遗项目卡片列表，每张卡片包含封面图、名称、分类、简介摘要
2. 支持按中国国家级非遗名录十大类筛选
3. 列表支持分页，每页默认显示 12 个项目
4. 页面在 3 秒内完成加载
5. 响应式布局适配桌面、平板、手机

### US-G02: 查看非遗项目详情 `[MVP]`
**作为** 访客，**我想要** 查看某个非遗项目的详细信息，**以便** 深入了解该项目的历史、传承和特色。

**验收标准：**
1. 详情页展示项目名称、分类、地域、历史背景、传承人信息、详细描述
2. 图片画廊支持缩略图预览和大图查看
3. 视频可在页面内嵌播放
4. 页面底部显示相关推荐项目
5. 支持通过 URL 直接访问特定项目详情页（SEO 友好）

### US-G03: 搜索非遗项目 `[MVP]`
**作为** 访客，**我想要** 通过关键词搜索非遗项目，**以便** 快速找到我感兴趣的内容。

**验收标准：**
1. 页面顶部提供搜索框，支持中英文关键词输入
2. 搜索结果按相关度排序展示
3. 搜索结果页显示匹配项目数量
4. 支持在搜索结果中进一步按分类筛选
5. 无结果时显示友好提示和推荐内容

### US-G04: 切换界面语言 `[MVP]`
**作为** 访客，**我想要** 在中文和英文之间切换界面语言，**以便** 用我熟悉的语言浏览网站。

**验收标准：**
1. 页面头部提供语言切换按钮（中/EN）
2. 切换后界面文本立即更新，无需刷新页面
3. 语言偏好保存在浏览器本地存储中
4. URL 路径包含语言标识（如 /zh/、/en/）

### US-G05: 查看非遗项目评价 `[MVP]`
**作为** 访客，**我想要** 查看其他用户对非遗项目的评价，**以便** 了解大家的看法和体验。

**验收标准：**
1. 项目详情页展示平均评分和评价总数
2. 评论列表按时间倒序排列，支持分页
3. 每条评论显示用户昵称、评分、评论内容、发布时间
4. 未登录用户可查看但不可发表评价

## 二、注册用户（User）故事

### US-U01: 注册账号 `[MVP]`
**作为** 新用户，**我想要** 使用邮箱注册账号，**以便** 获得评价、收藏、交流等功能权限。

**验收标准：**
1. 注册表单包含邮箱、密码、确认密码、昵称字段
2. 密码强度要求：至少 8 位，包含字母和数字
3. 注册后发送验证邮件，用户需点击链接激活账号
4. 邮箱已注册时显示明确提示
5. 表单验证错误实时提示

### US-U02: 登录账号 `[MVP]`
**作为** 注册用户，**我想要** 使用邮箱和密码登录，**以便** 访问我的个人功能。

**验收标准：**
1. 登录表单包含邮箱和密码字段
2. 登录成功后跳转到之前浏览的页面
3. 登录失败显示明确错误信息（邮箱或密码错误）
4. 支持"记住我"功能
5. 连续 5 次登录失败后临时锁定账号 15 分钟

### US-U03: 重置密码 `[MVP]`
**作为** 注册用户，**我想要** 在忘记密码时重置密码，**以便** 恢复账号访问。

**验收标准：**
1. 登录页提供"忘记密码"链接
2. 输入注册邮箱后发送密码重置邮件
3. 重置链接有效期为 24 小时
4. 重置成功后自动跳转到登录页
5. 非注册邮箱提交时不泄露该邮箱是否已注册

### US-U04: 对非遗项目评分和评论 `[MVP]`
**作为** 注册用户，**我想要** 对非遗项目进行评分和撰写评论，**以便** 分享我的看法和体验。

**验收标准：**
1. 项目详情页提供评分组件（1-5 星可点击选择）
2. 评论输入框支持文字输入，限制 500 字以内
3. 提交后评论立即显示在评论列表中
4. 同一用户对同一项目只能评分一次，但可修改
5. 提交前显示预览确认

### US-U05: 上传图片评论 `[P1]`
**作为** 注册用户，**我想要** 在评论中附带图片，**以便** 更直观地分享我的体验。

**验收标准：**
1. 评论表单支持上传最多 3 张图片
2. 支持 JPG、PNG 格式，单张不超过 5MB
3. 上传前显示图片预览
4. 图片自动压缩和生成缩略图
5. 评论中的图片可点击放大查看

### US-U06: 收藏非遗项目 `[MVP]`
**作为** 注册用户，**我想要** 收藏感兴趣的非遗项目，**以便** 日后方便查看。

**验收标准：**
1. 项目详情页和列表卡片上提供收藏按钮（心形图标）
2. 点击后立即切换收藏状态，无需刷新
3. 已收藏项目在个人中心的收藏夹中展示
4. 收藏夹支持取消收藏
5. 未登录用户点击收藏时引导登录

### US-U07: 管理个人资料 `[MVP]`
**作为** 注册用户，**我想要** 编辑我的个人资料，**以便** 展示我的身份信息。

**验收标准：**
1. 个人中心提供资料编辑页面
2. 可修改头像、昵称、个人简介
3. 头像支持上传图片，自动裁剪为正方形
4. 昵称唯一性校验
5. 修改成功后显示确认提示

### US-U08: 在评论区讨论 `[MVP]`
**作为** 注册用户，**我想要** 在非遗项目评论区参与讨论，**以便** 与其他用户交流看法。

**验收标准：**
1. 项目详情页底部显示评论区
2. 支持对其他用户的评论进行回复
3. 回复以嵌套形式展示（最多 2 层）
4. 评论支持按时间和热度排序
5. 自己的评论可以删除

### US-U09: 参与社区论坛 `[P1]`
**作为** 注册用户，**我想要** 在社区论坛发帖和回帖，**以便** 与同好深入交流非遗话题。

**验收标准：**
1. 论坛按非遗类型分区展示
2. 可创建新帖子，包含标题和正文
3. 可对帖子进行回复
4. 帖子列表按最新回复时间排序
5. 自己的帖子可以编辑和删除

### US-U10: 查看我的评论和帖子历史 `[P1]`
**作为** 注册用户，**我想要** 查看我发表过的所有评论和帖子，**以便** 追踪我的参与记录。

**验收标准：**
1. 个人中心提供"我的评论"和"我的帖子"标签页
2. 按时间倒序展示
3. 点击可跳转到对应的项目详情页或帖子
4. 支持分页浏览

### US-U11: 查看浏览历史 `[P2]`
**作为** 注册用户，**我想要** 查看我的浏览历史，**以便** 找回之前看过的非遗项目。

**验收标准：**
1. 个人中心提供浏览历史页面
2. 按时间倒序展示最近浏览的项目
3. 支持清除单条或全部历史
4. 最多保留最近 100 条记录

## 三、内容管理员（Content Manager）故事

### US-C01: 创建非遗项目 `[MVP]`
**作为** 内容管理员，**我想要** 创建新的非遗项目条目，**以便** 将非遗信息录入平台。

**验收标准：**
1. 后台提供非遗项目创建表单
2. 必填字段：名称、分类（十大类）、地域、简介、详细描述
3. 选填字段：传承人信息、历史背景、相关链接
4. 支持富文本编辑器编写详细描述
5. 创建成功后可选择"保存为草稿"或"直接发布"

### US-C02: 编辑非遗项目 `[MVP]`
**作为** 内容管理员，**我想要** 编辑已有的非遗项目信息，**以便** 更新和完善内容。

**验收标准：**
1. 项目列表页提供编辑入口
2. 编辑表单预填充现有数据
3. 修改后可保存为草稿或直接发布
4. 保存成功后显示确认提示
5. 支持从已发布状态改为下架

### US-C03: 管理多媒体资源 `[MVP]`
**作为** 内容管理员，**我想要** 为非遗项目上传和管理图片、视频、音频，**以便** 丰富项目展示内容。

**验收标准：**
1. 项目编辑页提供多媒体上传区域
2. 图片支持批量上传，格式：JPG/PNG/WebP，单张不超过 10MB
3. 视频支持上传，格式：MP4/WebM，单个不超过 500MB
4. 音频支持上传，格式：MP3/WAV，单个不超过 50MB
5. 可对已上传的媒体进行排序、删除、设置封面图

### US-C04: 管理非遗项目列表 `[MVP]`
**作为** 内容管理员，**我想要** 查看和管理所有非遗项目，**以便** 掌握内容整体状态。

**验收标准：**
1. 后台提供项目列表页，显示名称、分类、状态、创建时间
2. 支持按分类、状态（草稿/已发布/已下架）筛选
3. 支持按名称搜索
4. 支持批量操作（发布、下架）
5. 列表支持分页

### US-C05: 管理用户评论 `[MVP]`
**作为** 内容管理员，**我想要** 查看和管理用户评论，**以便** 维护平台内容质量。

**验收标准：**
1. 后台提供评论管理列表
2. 显示评论内容、评论者、关联项目、发布时间
3. 支持删除不当评论
4. 支持按项目筛选评论
5. 删除评论时记录操作日志

### US-C06: 录入双语内容 `[P1]`
**作为** 内容管理员，**我想要** 为非遗项目录入中英双语内容，**以便** 国际用户也能理解。

**验收标准：**
1. 项目编辑表单提供中文和英文两个标签页
2. 中文为必填，英文为选填
3. 切换标签页时保留已填写内容
4. 未填写英文内容时，前台英文版显示中文原文

## 四、系统管理员（Admin）故事

### US-A01: 管理用户账号 `[MVP]`
**作为** 系统管理员，**我想要** 查看和管理所有用户账号，**以便** 维护平台用户秩序。

**验收标准：**
1. 后台提供用户列表页，显示昵称、邮箱、角色、注册时间、状态
2. 支持按角色、状态筛选
3. 支持按昵称或邮箱搜索
4. 可禁用/启用用户账号
5. 禁用后用户无法登录，已登录会话立即失效

### US-A02: 分配用户角色 `[MVP]`
**作为** 系统管理员，**我想要** 为用户分配角色，**以便** 控制不同用户的权限。

**验收标准：**
1. 用户详情页提供角色分配功能
2. 可将用户角色设置为：注册用户、内容管理员、系统管理员
3. 角色变更立即生效
4. 不能修改自己的角色
5. 角色变更记录操作日志

### US-A03: 查看内容管理仪表板 `[MVP]`
**作为** 系统管理员，**我想要** 查看平台内容概览，**以便** 了解平台运营状态。

**验收标准：**
1. 仪表板显示：非遗项目总数、用户总数、评论总数、今日访问量
2. 显示最近创建/更新的非遗项目
3. 显示最近注册的用户
4. 显示待处理事项（如被举报的评论）
5. 数据自动刷新或提供手动刷新按钮

### US-A04: 查看数据统计 `[P1]`
**作为** 系统管理员，**我想要** 查看详细的数据统计，**以便** 分析平台使用情况和趋势。

**验收标准：**
1. 提供访问量趋势图（日/周/月）
2. 显示热门非遗项目排行（按浏览量、收藏量）
3. 显示用户活跃度统计
4. 支持按时间范围筛选
5. 支持数据导出（CSV 格式）

### US-A05: 管理系统配置 `[P1]`
**作为** 系统管理员，**我想要** 配置系统参数，**以便** 调整平台行为。

**验收标准：**
1. 提供系统配置页面
2. 可配置：网站名称、Logo、每页显示数量、文件上传大小限制
3. 配置修改后立即生效
4. 提供配置项说明文字
5. 关键配置修改需二次确认

### US-A06: 管理论坛内容 `[P1]`
**作为** 系统管理员，**我想要** 管理社区论坛的帖子和回复，**以便** 维护社区秩序。

**验收标准：**
1. 后台提供论坛帖子管理列表
2. 可删除违规帖子和回复
3. 可置顶/取消置顶帖子
4. 支持按分区筛选
5. 删除操作记录日志
