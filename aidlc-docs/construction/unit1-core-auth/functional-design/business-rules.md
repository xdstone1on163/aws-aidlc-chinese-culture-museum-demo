# Unit 1: 后端核心与认证 - 业务规则

## BR-01: 密码策略

| 规则 | 说明 |
|------|------|
| 最小长度 | 8 位 |
| 复杂度 | 必须包含字母和数字 |
| 存储 | Django 内置 PBKDF2 哈希 |
| 历史密码 | MVP 不限制 |

---

## BR-02: 注册规则

| 规则 | 说明 |
|------|------|
| 邮箱唯一 | 同一邮箱不可重复注册，已注册时返回明确提示 |
| 昵称唯一 | 全局唯一，注册时校验 |
| 邮箱验证 | 注册后发送验证邮件，用户需点击链接激活 |
| 验证链接有效期 | 24 小时 |
| 未验证状态 | 可登录但功能受限（不可评论、收藏） |
| 默认角色 | user（注册用户） |
| 默认语言 | zh（中文） |

---

## BR-03: 登录规则

| 规则 | 说明 |
|------|------|
| 登录标识 | 邮箱 + 密码 |
| Token 类型 | JWT（Access Token + Refresh Token） |
| Access Token 有效期 | 1 小时 |
| Refresh Token 有效期 | 30 天 |
| "记住我" | 勾选时 Refresh Token 有效期延长至 30 天，否则为 7 天 |
| 未验证邮箱 | 允许登录，返回 is_verified=false 标记 |
| 被禁用账号 | 拒绝登录，返回"账号已被禁用"提示 |

---

## BR-04: 登录失败锁定（Redis）

| 规则 | 说明 |
|------|------|
| 计数器 Key | `login_fail:{email}` |
| 最大失败次数 | 5 次 |
| 锁定时长 | 15 分钟 |
| 计数器过期 | 15 分钟（与锁定时长一致） |
| 成功登录 | 清除计数器 |
| 锁定期间 | 返回"账号已临时锁定，请15分钟后重试" |

**流程**：
1. 登录请求 → 检查 Redis 中 `login_fail:{email}` 的值
2. 值 ≥ 5 → 拒绝登录，返回锁定提示
3. 值 < 5 → 验证密码
4. 密码错误 → INCR 计数器，设置 TTL 15分钟
5. 密码正确 → DEL 计数器，签发 Token

---

## BR-05: 密码重置规则

| 规则 | 说明 |
|------|------|
| 触发方式 | 用户提交注册邮箱 |
| 安全策略 | 无论邮箱是否存在，都返回"如果邮箱已注册，将收到重置邮件"（防枚举） |
| 重置链接有效期 | 24 小时 |
| 一次性使用 | 链接使用后立即失效 |
| 重置成功 | 所有已签发的 Token 失效（通过 Redis 黑名单） |

---

## BR-06: 用户禁用规则（Redis Token 黑名单）

| 规则 | 说明 |
|------|------|
| 禁用操作 | Admin 将 user.is_active 设为 False |
| Token 失效 | 将用户 ID 加入 Redis 黑名单 Key `user_disabled:{user_id}`，TTL = Access Token 最大有效期 |
| 请求校验 | JWT 认证中间件在验证 Token 后，检查 Redis 黑名单 |
| 启用操作 | Admin 将 user.is_active 设为 True，删除 Redis 黑名单 Key |

---

## BR-07: 角色管理规则

| 规则 | 说明 |
|------|------|
| 角色变更权限 | 仅 admin 可操作 |
| 自我保护 | admin 不能修改自己的角色 |
| 立即生效 | 角色变更后，下次 Token 刷新时生效 |
| 操作日志 | 角色变更记录到 Django Admin LogEntry |

---

## BR-08: 公共规则（core app）

### 分页
- 默认每页 12 条（列表页）/ 20 条（管理页）
- 最大每页 100 条
- 使用 PageNumberPagination

### API 响应格式
```json
{
  "code": 200,
  "message": "success",
  "data": { ... }
}
```

### 错误响应格式
```json
{
  "code": 400,
  "message": "错误描述",
  "errors": { "field": ["错误详情"] }
}
```

### 速率限制（Redis）
- 匿名用户：60 次/分钟
- 认证用户：120 次/分钟
- 登录接口：10 次/分钟（防暴力破解）
