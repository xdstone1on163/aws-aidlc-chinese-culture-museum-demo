# Unit 2: 媒体管理 - 业务规则

## BR-M01: 文件类型限制

| 文件类型 | 允许格式 | 最大大小 |
|----------|----------|----------|
| 图片 | JPG, PNG, WebP | 10MB |
| 视频 | MP4, WebM | 500MB |
| 音频 | MP3, WAV | 50MB |

## BR-M02: 上传规则

| 规则 | 说明 |
|------|------|
| 认证要求 | 必须登录 |
| 权限 | 内容管理员可上传非遗项目媒体；注册用户可上传头像和评论图片 |
| 批量上传 | 图片支持批量上传（单次最多 10 张） |
| 文件名 | 存储时使用 UUID 重命名，保留原始文件名在元数据中 |
| 存储路径 | `media/{object_type}/{YYYY}/{MM}/{uuid}.{ext}` |

## BR-M03: 封面图规则

| 规则 | 说明 |
|------|------|
| 每个对象最多 1 张封面图 | 设置新封面时自动取消旧封面 |
| 仅图片可设为封面 | 视频和音频不可设为封面 |

## BR-M04: 删除规则

| 规则 | 说明 |
|------|------|
| 删除元数据 | 同时删除本地文件 |
| 级联删除 | 关联对象删除时，清理所有相关媒体文件 |
